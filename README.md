# 🚀 Crypto Alert Telegram Bot

Telegram-бот для мониторинга курсов криптовалют в реальном времени. Позволяет пользователям устанавливать уведомления (алерты) на изменение цены конкретных активов.

## ✨ Ключевые возможности
* **Мониторинг цен:** Интеграция с CoinGecko API для получения актуальных курсов.
* **Персональные уведомления:** Создание алертов на достижение целевой цены через команду `/alert`.
* **Управление подписками:** Просмотр и удаление активных уведомлений через интерактивное меню `/my_alerts`.
* **Фоновые задачи:** Асинхронная проверка цен без блокировки работы бота.
* **Надежность:** Использование PostgreSQL для хранения данных пользователей и их настроек.

## 🛠 Стек технологий
* **Language:** Python 3.10+
* **Framework:** [aiogram 3.x](https://github.com/aiogram/aiogram) (Asynchronous Telegram Bots)
* **Database:** PostgreSQL + [SQLAlchemy 2.0](https://www.sqlalchemy.org/) (ORM)
* **API:** CoinGecko API (aiohttp)
* **Containerization:** Docker & Docker-compose

## 📋 Команды бота
* `/start` — Регистрация и приветствие.
* `/alert [coin] [price]` — Установить уведомление (например, `/alert btc 65000`).
* `/my_alerts` — Список ваших активных уведомлений с возможностью удаления.

## 🚀 Быстрый запуск (Docker)

Проект полностью контейнеризирован. Для запуска вам потребуется только установленный Docker.

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/RB2678/CryptoMonitoringBot.git
   cd CryptoMonitoringBot

2. **Настройте переменные окружения:**
   ```bash
   cp .env.example .env
   # Отредактируйте .env, добавив свой BOT_TOKEN и параметры БД

3. **Запустите проект:**
   ```bash
   docker-compose up -d --build
  Бот автоматически инициализирует базу данных и начнет работу.

## ⚙️ Конфигурация (.env)

В проекте используются следующие ключевые переменные:

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Ваш уникальный токен от **@BotFather** |
| `POSTGRES_USER` | Имя пользователя БД |
| `POSTGRES_PASSWORD` | Пароль пользователя БД |
| `POSTGRES_DB` | Название базы данных |
| `DATABASE_URL` | Строка подключения (формируется на основе данных выше) |
